# MIGRANT ID Work Visa Readiness Test - English Bitrix Version

English version of the MIGRANT ID work visa readiness test for international candidates who use English, including applicants from India, Pakistan, the Philippines and other markets.

## Main Files

The working local project is stored on the computer in:

`C:\Users\wallc\Documents\Сайт и реклама\visa-readiness-test-eng-bitrix`

Key local files:

- `visa-readiness-test.html` - source HTML version for development and local checks.
- `visa-readiness-test-bitrix.html` - Bitrix-ready HTML with the logo embedded in the code.
- `BITRIX-CODE-PASTE-ME.txt` - same Bitrix-ready code in a text file for easy copy-paste into a Bitrix HTML block.
- `google-sheets-apps-script.js` - Apps Script handler used by the test to save submissions to Google Sheets.

## Bitrix Publishing

1. Open `BITRIX-CODE-PASTE-ME.txt` from the local project folder.
2. Copy the whole file content.
3. Paste it into a Bitrix HTML block.
4. Save and publish the page.
5. Check the page on mobile and desktop after clearing cache or adding a version parameter such as `?v=en1`.

## Current Logic

- The test gives a preliminary readiness score, not a visa guarantee.
- Stop factors are shown separately for critical issues such as an unsuitable work language or criminal record where a police clearance is required.
- Candidates may select several languages; the score uses the strongest relevant language for the selected country.
- MIGRANT ID is treated as a profile/employer-access booster, not as a standard visa document.
