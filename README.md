# Migrant ID Visa Readiness Test

Интерактивный тест визовой готовности для кандидатов Migrant ID.

## Что делает тест

- собирает базовые данные кандидата;
- формирует список документов под страну, срок и тип вакансии;
- считает предварительный процент визовой готовности;
- показывает стоп-факторы и недостающие обязательные пункты;
- сохраняет ответы в Google Sheets;
- переводит кандидата на страницу связи со специалистом Migrant ID.

## Основные файлы

- `visa-readiness-test.html` - рабочий исходный файл теста.
- `visa-readiness-test-bitrix.html` - готовая версия для вставки в HTML-блок Bitrix.
- `visa-readiness-test-bitrix-COPY-CODE.txt` - тот же код, но в формате для удобного копирования.
- `migrant-id-logo.png` - логотип.
- `google-sheets-apps-script.js` - обработчик для сохранения ответов в Google Sheets.
- `google-sheets-setup.md` - инструкция по Google Sheets.
- `AGENTS.md` - постоянные правила для AI-агента и дизайн-бриф проекта.

## Публикация в Bitrix

1. Открыть `visa-readiness-test-bitrix-COPY-CODE.txt`.
2. Скопировать весь код.
3. Вставить в HTML-блок Bitrix.
4. Сохранить и опубликовать страницу.
5. Проверить страницу в режиме инкогнито или с параметром `?v=2`.

## Текущая логика языков

Кандидат может выбрать несколько языков. Для процента готовности учитывается не сумма всех языков, а лучший релевантный язык для выбранной страны:

- язык выбранной страны;
- английский;
- русский только для части стран Восточной Европы и с меньшим весом.

Если ни один выбранный язык не закрывает рабочую коммуникацию для страны, тест показывает стоп-фактор.
